📐 Formulario — Leggi dei Circuiti

Legge di Ohm

V = R · I
V: tensione [V]
R: resistenza [Ω]
I: corrente [A]
P = V·I = R·I² = V²/R

Kirchhoff

KCL (nodi): Σ Iₖ = 0

KVL (maglie): Σ Vₖ = 0
Somma d.d.p. in maglia chiusa = 0

Condensatore

Q = C·V, I = C·dV/dt
Energia: U = ½CV²
Carica CC: V(t)=ε(1−e^−t/τ)
τ = RC
Serie: 1/C_eq=Σ1/Cₖ

Induttore

V = L·dI/dt
Energia: U = ½LI²
Carica CC: I(t)=(ε/R)(1−e^−t/τ)
τ = L/R
Serie: L_eq=ΣLₖ

Impedenza (CA)

Z_R = R
Z_C = 1/(jωC) → X_C=1/(ωC)
Z_L = jωL → X_L=ωL
|Z|=√(R²+(X_L−X_C)²)
φ=arctan((X_L−X_C)/R)

Risonanza RLC

ω₀ = 1/√(LC)
f₀ = ω₀/(2π)
Q: (1/R)√(L/C)
A risonanza: Z=R (minimo)
I = ε/R (massimo)

f.e.m. e potenza

ε = V_t + r·I
r: resistenza interna
P_tot = ε·I
P_Joule = R·I²
Rendimento: η=P_u/P_tot

Serie / Parallelo

R serie: R_eq=ΣRₖ
R parall.: 1/R_eq=Σ1/Rₖ
C serie: 1/C_eq=Σ1/Cₖ
C parall.: C_eq=ΣCₖ
L serie: L_eq=ΣLₖ
